# Feedback Hub

A simple web-based feedback management system that allows users to submit feedback and administrators to view and manage responses. The project is built using HTML, CSS, JavaScript, and Firebase for data storage.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Features
- Submit feedback through an easy-to-use interface
- Store feedback securely in Firebase
- Admin dashboard to view submitted feedback
- Real-time database integration
- Responsive and user-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Firebase

##  Project Structure
```
feedback_hub/
│
├── index.html            # User feedback form
├── script.js             # Feedback submission logic
├── admin.html            # Admin dashboard
├── admin.js              # Admin functionality
├── firebase-config.js    # Firebase configuration
└── README.md             # Project documentation
```
##  Setup Instructions
### 1. Clone the Repository

```bash
git clone https://github.com/Ezilarasi2006/feedback_hub.git
```
### 2. Open the Project
```bash
cd feedback_hub
```
### 3. Configure Firebase
- Create a Firebase project.
- Enable the required Firebase services.
- Update `firebase-config.js` with your Firebase credentials.

### 4. Run the Project
Open `index.html` in your browser.
